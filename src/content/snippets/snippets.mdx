# 코드 스니펫

제가 자주 사용하는 코드 조각들입니다.

## Vite + Prettier + ESLint 프로젝트 시작

```bash
npm create vite@latest .
npm i
npm i -D eslint prettier eslint-plugin-prettier eslint-config-prettier eslint-plugin-react @typescript-eslint/parser @typescript-eslint/eslint-plugin
```

## Prettier 설정

```json
{
	"semi": true,
	"singleQuote": false,
	"endOfLine": "lf",
	"singleAttributePerLine": true,
	"bracketSameLine": true,
	"trailingComma": "none"
}
```
